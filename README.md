# vschat/api

The powerful backend powering **VSChat** - the free and fast **[Visual Studio Code](https://code.visualstudio.com)** chat app.

## Technology Stack

- **TypeScript** - Main language
- **Express** - Web server framework
- **PostgreSQL** - Database system
- **Prisma** - ORM
